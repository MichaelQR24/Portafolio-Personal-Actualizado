Sistema Barbería

Gestión de citas & clientes

Sistema de Gestión para Barbería

2024

Aplicación web completa para gestión de citas, clientes y servicios en barberías

  • Problema

    Las barberías tradicionales gestionaban citas manualmente con agendas físicas, causando conflictos de horarios, pérdida de información y dificultad para analizar el rendimiento del negocio.

  • Solución

    Desarrollé un sistema web integral con Java y MySQL que permite reservar citas online, gestionar clientes, servicios y generar reportes automáticos. Interfaz intuitiva para barberos y clientes.
